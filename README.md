# POC LangChain Agent

This is a [Next.js](https://nextjs.org) project demonstrating a Cognitive Agent using LangChain and LangGraph.

## Features

- **Cognitive Agent:** Built with LangGraph.
- **Tools:** Modular tools system (e.g., Calculator).
- **Security:** "Git Guardians" (Secretlint, Husky) to prevent leaking secrets.
- **CI/CD:** GitHub Actions for automated testing and validation.
